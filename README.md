已提交组件的简介:
=================
1.carousel
走马灯，一个jQuery的开源组件，适用于pc移动端，可以根据屏比的变化调整展示item个数。

2.ios组件
作为IOS的一些组件的效果用例，比如h1,a, btn,radiobox,checkbox,锁屏……
利用html5,CSS3构建IOS的一些茶用组件。

3.path-menu
快捷导航菜单，这是改的一个html5/css3的开源组件，解决了PC端和移动端的各种浏览器对html5/css3部分属性的的兼容性问题。
